# EchoReply

> Understand the email. Then reply.

EchoReply is a Manifest V3 Chrome extension that injects an AI-powered sidebar into Gmail — decoding the sender's real intent, emotional temperature, and reply strategy, then drafting replies in three tones (🎩 Professional, 🤝 Warm, ⚡ Confident).

---

## 📥 Download / Install

**👉 [Install from the Chrome Web Store](https://chromewebstore.google.com/detail/leibhmkahdofhfallekeoljmiajcjjlk)** — one-click install, auto-updates.

Prefer to load it manually? **[Download echoreply.zip](https://github.com/HAONANTAO/EchoReply/raw/main/echoreply.zip)** — unzip, then follow the Developer Mode steps in [`echoreply/README.md`](echoreply/README.md).

---

## What it does

For any open Gmail thread, the sidebar shows:

1. **Real Intent** — what the sender actually wants
2. **Emotional Temperature** — `WARM` / `COOL` / `NEUTRAL` / `URGENT` / `FRUSTRATED`
3. **Reply Strategy** — `MUST_REPLY` / `SHOULD_REPLY` / `OPTIONAL` / `DONT_REPLY`
4. **Three reply drafts** in different tones, one click to insert into Gmail's compose box
5. **Tone Check** — paste your own draft, get flagged phrases and a natural rewrite

You bring your own Claude or OpenAI API key. No backend, no telemetry — your key and the emails you analyze stay between your browser and the provider you choose.
